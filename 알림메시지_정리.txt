🗑️ 삭제 확인 모달 (deleteConfirmModal)
===============================================

📋 모달 제목
- "대시보드 삭제 확인"

📝 모달 내용
- 아이콘: ⚠️ (fas fa-exclamation-triangle)
- 메인 메시지: "정말로 이 대시보드를 삭제하시겠습니까?"
- 대시보드 이름: "${deleteDashboardName}" (동적)
- 경고 메시지: "삭제된 대시보드는 복구할 수 없습니다."

🔘 버튼
- "취소" - 모달 닫기
- "삭제" - 삭제 실행

===============================================
📝 참고사항
===============================================
- ✅ = 성공 메시지 (초록색)
- ❌ = 오류/경고 메시지 (빨간색)
- ${변수명} = 동적으로 변하는 값
- 모든 알림은 3초 후 자동으로 사라집니다.
- 알림 위치: 화면 우상단 (top: 80px, right: 20px)

🔔 컨펌창 종류
- JavaScript confirm(): 브라우저 기본 확인 대화상자
- JavaScript alert(): 브라우저 기본 알림 대화상자
- 커스텀 모달: 삭제 확인 모달 (deleteConfirmModal)
